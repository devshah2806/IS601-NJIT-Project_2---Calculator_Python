# calculator_python

Calculator using Python that has automated unit tests.
* Basic Calculation,
  * Addition
  * Subtraction
  * Multiplication
  * Division
  * Squares
  * Square Root
* Statistical Functions,
  * Mean
  * Median
  * Mode
  * Standard Deviation
  * Variance

* Using correct
  * Static Methods
  * Object Methods
  * Object Attributes/ Properties
  * Random Generators
